    Movie Recommendation

    • We use RowMatrix.columnSimilarities to compute similarity between movies
    • We define crew title importance as following
    "actor" => 5
"actress" => 5
"director" => 20
"writer" => 15
"producer" => 10
    • We use crew title importance to construct movie vector
    For example, if one movie has crew_0 as *actress* and crew_3 as *director*, its vector will be [5, 0, 0, 20]
    • We use The Godfather and The Avengers as Evaluation inputs

    Compute similarity using cosine similarity
    %spark
    import org.apache.spark.mllib.linalg.distributed.MatrixEntry
    import org.apache.spark.mllib.linalg.distributed.CoordinateMatrix
    // indexed -> matrixEntry -> CoordinateMatrix -> RowMatrix -> ComputeSimilarity
    val matEntries = indexed
    .dropDuplicates("nconstIndex", "tconstIndex")
    .select($"nconstIndex", $"tconstIndex", transformPrincipalCategory($"category"))
    .rdd
    .map(row=>{
    val i = row.getDouble(0)
    val j = row.getDouble(1)
    val value = row.getInt(2)
    MatrixEntry(i.toLong, j.toLong, value)
    })
    val rowMat = new CoordinateMatrix(matEntries)
    .toRowMatrix
    val similarity = rowMat.columnSimilarities(0.05)

    Comments on the Evaluation

    • In the first similarity evaluation of the movie The Godfather, we can see that all three movies from the godfather series are included in the top 3 results
    • In the second similarity evaluation of the movie The Avengers, Avengers: Age of Ultron is the only movie from the avenger series and it’s not the first match. The reason why other movie ranked higher is because cosine distance normalized the similarity score with the vector module, which makes the movie with more crew like Avengers: Age of Ultron be ranked lower and movie with few crew like Untitled Joss Whedon ranked first.

    In the next section, we will add vector module into consideration and evaluate the results again.

    Module compute
    %spark
    // compute movie vector module
    // vector module = sqrt(x1^2+x2^2+...)
    val movieVectorModule = indexed
    .select("tconstIndex", "nconstIndex", "category")
    .dropDuplicates("tconstIndex", "nconstIndex")
    .withColumn("score", transformPrincipalCategory($"category"))
    // compute module
    .withColumn("scoreSquare", pow($"score", 2))
    .groupBy("tconstIndex")
    .agg(sqrt(sum("scoreSquare")).as("module"))
    movieVectorModule: org.apache.spark.sql.DataFrame = [tconstIndex: double, module: double]
